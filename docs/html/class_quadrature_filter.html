Detailed Description

A quadrature filter pair.

An implementation of the quadrature pair filter described here.

Processes the input to create two output signals with the same magnitude spectrum as the input but phase spectra which differ by \( \frac{\pi}{2} \) radians for a large portion of the frequency range.

Similar to Max/MSP's hilbert~ object.
